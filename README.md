# Curio-agent Project

This project is an event aggregation and recommendation system for ANU students.

## Project Structure

- **[frontend/](frontend/)**: The Next.js frontend application.
- **[server-python/](server-python/)**: Python FastAPI backend for AI recommendations and scraping.
- **[server-nestjs/](server-nestjs/)**: NestJS backend for core business logic and task scheduling.

## Getting Started

Refer to the README in each directory for specific setup instructions.
